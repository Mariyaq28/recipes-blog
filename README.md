# Recipe Blog WebApp (RICETTE DELIZIOSE)
 
A simple role-based recipe blog built with **Flask** and **SQLite**.

## 🔧 Features
- **Admin Dashboard**: Upload, manage, and delete recipes.
- **User Login/Profile**: Like and save favorite recipes.
- **Recipe Categories**: Browse by type (Drinks, Sweets, etc.).
- **Search Functionality**: Find recipes by title.
- **Authentication**: Role-based access control.

## 🛠 Tech Stack
- Flask + Jinja2  
- SQLite (SQLAlchemy ORM)  
- HTML/CSS + Bootstrap (optional)

## 🚀 Run Locally
```bash
git clone https://github.com/Rida3535/RecipeBlog
python -m venv env
source env/bin/activate  # or env\Scripts\activate on Windows
pip install -r requirements.txt
flask run
```
